# Personlig Portfölj - Vite & React

En modern och högpresterande portföljhemsida byggd med de senaste webbteknologierna.

## Mål med projektet
Målet var att bygga en snabb, responsiv och visuellt tilltalande portfölj som speglar min tekniska profil. Genom att använda `Vite` och `React` säkerställdes minimala laddningstider och en smidig utvecklarupplevelse, medan `TypeScript` användes för att garantera kodkvalitet och skalbarhet.

## Kompetenser som testas
- **Frontend-utveckling:** Avancerad användning av `React` för komponentbaserad arkitektur.
- **Modern Tooling:** Konfiguration av `Vite` för snabb bundling och optimerad produktionskod.
- **TypeScript:** Typ-säker utveckling för att eliminera vanliga buggar och förbättra dokumentationen av koden.
- **Deployment & CI/CD:** Hantering av driftsättning via plattformar som `Netlify` eller `Cloudflare Pages`, samt integration med moderna byggflöden.
- **UX/UI-design:** Fokus på responsivitet och interaktionsdesign för att skapa en bra användarupplevelse på alla enheter.

## Installation & Utveckling
Projektet använder `bun` eller `npm`.
```bash
npm install
npm run dev
```

## Bygg för produktion
```bash
npm run build
```
Resultatet hamnar i `dist/`-mappen och är redo för hosting.
